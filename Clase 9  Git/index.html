Comandos de git

git init => inicializar repositorio

git add . => Agregar archivos a stagin area/lista de espera

git commit -m "comentario" => Crear una nueva version de nuestro repositorio

git status =. ver el estado de nuestro repositorio y sus archivos

git branch =. Ver nuestras ramas y la rama en la que estamos

git checkout -b nombre-de-la-rama =. Crear nueva rama

git checkout nombre-de-la-rama => Movernos hacia una rama

git merge nombre-de-la-rama => Traer los cmabios de una rama
